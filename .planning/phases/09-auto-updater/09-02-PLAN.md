<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| GitHub Releases HTTPS → Rust plugin | latest.json fetched over TLS; signature verified by plugin against embedded pubkey |
| JS renderer → Tauri IPC | check() call requires updater:default capability — blocked by default without explicit grant |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-9-02-01 | Tampering | latest.json endpoint (GitHub Releases) | mitigate | HTTPS enforced by tauri-plugin-updater; `.sig` file verified against `pubkey` in tauri.conf.json before install; unsigned artifacts rejected |
| T-9-02-02 | Tampering | MITM on latest.json fetch | mitigate | Ed25519 signature on artifact + HTTPS for transport; even if latest.json is spoofed, artifact without valid `.sig` is rejected |
| T-9-02-03 | Tampering | Downgrade attack (serve older version) | accept | Tauri updater only installs versions > current semver; serving an older version causes check() to return null |
| T-9-02-04 | Denial of Service | Network failure during check() | mitigate | Plan 09-03 wraps check() in try/catch; network failure is silent (UPD-07); no crash, no retry storm |
| T-9-02-05 | Tampering | installMode set to "quiet" (requires admin UAC) | mitigate | Explicitly set `"installMode": "passive"` — no interactive wizard, no UAC prompt escalation |
</threat_model>
