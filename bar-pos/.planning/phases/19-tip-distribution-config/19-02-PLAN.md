<objective>
Build the database heart of Phase 19: a new append-only tip_distribution_entries table and an in-place extension of the existing close_caja_session RPC that, atomically with the caja close, reads the current tip-distribution config, pools the session's tips, splits them via largest-remainder, and writes one immutable entry row.
