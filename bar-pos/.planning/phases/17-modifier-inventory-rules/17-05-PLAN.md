<objective>
Ship the admin UI (SC-3): a per-modifier "Ingredient rules" dialog inside the Settings → Products → Modifiers surface, letting managers attach N ingredient rules with signed deltas to any modifier. The dialog is a new feature slice; `CatalogModifiersTab.tsx` gains a per-row button that opens it.

Purpose: This is the operator-facing half of the phase — without it, rules can only be set via SQL. The delta field MUST be a plain signed numeric input (D-02), never `MoneyInput` (which clamps negatives to 0).

Output: `features/manage-modifier-inventory-rules/` slice (dialog + save hook + barrel); `CatalogModifiersTab.tsx` wired with the button + dialog; human-verified UAT.
</objective>
